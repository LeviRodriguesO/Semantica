Semântica

O que é semântica?

É o estudo do significado. É como entendemos e se comunicamos com a língua humana.

Para o HTML 5, a semântica é a descrição clara do seu significado de uma maneira que seja legível para humanos e máquinas.

<HEADER>

Usado para definir o cabeçalho da página e introduzir o assunto da página ou de um artigo, geralmente é a primeira tag do site. 


        <!DOCTYPE html>
        <html lang="pt-br">
        <head>
        <meta charset="UTF-8">
        <meta http-equiv="X-UA-Compatible" content="IE=edge">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
         <title>Semântica</title>
         <body>
            <header>
            <h1>Vai na Web</h1>
            </header>
         </body>
         </head>

<nav>

Representa uma lista de links internos.


        <!DOCTYPE html>
        <html lang="pt-br">
        <head>
        <meta charset="UTF-8">
        <meta http-equiv="X-UA-Compatible" content="IE=edge">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
         <title>Semântica</title>
         <body>
            <main>
             <nav>
              <a href="#about">Sobre Mim</a>
              <a href="#history">Minha História</a>
              <a href="#skills">Habilidades</a>
             </nav>
            </main>
         </body>
         </head>

<main>

Define o conteúdo principal da página, deve ser única


        <!DOCTYPE html>
        <html lang="pt-br">
        <head>
        <meta charset="UTF-8">
        <meta http-equiv="X-UA-Compatible" content="IE=edge">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
         <title>Semântica</title>
         <body>
            <main>
              TODO O CONTEÚDO PRINCIPAL DA PÁGINA.
            </main>
         </body>
         </head>

<section>

Representa uma seção dentro do seu site. 


        <!DOCTYPE html>
        <html lang="pt-br">
        <head>
        <meta charset="UTF-8">
        <meta http-equiv="X-UA-Compatible" content="IE=edge">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
         <title>Semântica</title>
         <body>
            <main>
              <section>
              <h1<Sobre Mim>
              <p>Eu começei minha carreira no ano de 1990.</p>
              </section>
            </main>
         </body>
         </head>

<figure> e <figcaption>

  • figure: Serve para definir uma foto, ilustração, diagrama e etc.
  • figcaption: Serve como uma legenda associada com a figura ou ilustração.

<article>

  • Article: Tag usada para definir como artigo o conteúdo da página.

<aside>

  • aside: A tag define um conteúdo ao lado de um artigo.

<div>

  • div:Define uma divisão ou uma seção do documento e ocupa toda a largura (elemento nível de bloco).

<span>

  • span: Usada para agrupar textos ou tags no documento e ocupa a largura necessária.

<footer>

  • footer: Usado para definir o rodapé da página.
